Presupuesto - Plataforma Diamont Deals
Descripción del Proyecto
Desarrollo de plataforma web para la gestión profesional de jugadores de poker con sistema automatizado de cálculo de comisiones (rakeback) basado en rendimiento individual.
Alcance del Proyecto
Panel de Administración
Gestión completa de jugadores
Gestión de clubs de poker
Sistema de condiciones dinámicas configurables
Generación automática de códigos de registro únicos
Dashboard con métricas y estadísticas
Portal de Jugadores
Registro mediante código único proporcionado
Dashboard personal con estadísticas de rendimiento
Visualización de comisiones por club
Historial de estadísticas mensuales
Gestión de perfil personal
Sistema de Cálculo Automático
Procesamiento automático de comisiones según tabla de condiciones
Aplicación de reglas basadas en ratio resultado/rake
Gestión multi-club por jugador
Histórico de cálculos y auditoría

Cronograma
Primera Semana (Días 1-5)
Días 1-2: Configuración de infraestructura y sistema de autenticación
Días 3-4: Desarrollo del panel de administración
Día 5: Sistema de condiciones y portal de jugadores

Segunda Semana (Días 6-12)
Día 6-7: Implementación de lógica de cálculo
Día 8: Testing integral
Día 9: Despliegue en producción
Día 10-12: Margen para ajustes y revisiones finales

Especificaciones Técnicas
Stack Tecnológico
Frontend: Next.js 14 con TypeScript
Base de datos: PostgreSQL 
Autenticación: Supabase Auth
Hosting: Vercel
Características de Seguridad
Autenticación robusta con tokens JWT
Políticas de seguridad a nivel de base de datos (RLS)
Encriptación de datos sensibles
Backup automático diario 
Funcionalidades Detalladas
Módulo de Jugadores
Generación automática de códigos únicos de registro
Estados: pendiente, activo, inactivo
Vinculación a múltiples clubs simultáneamente
Histórico completo de actividad
Sistema de Condiciones Dinámicas
Configuración de reglas basadas en rendimiento:
Ratio resultado/rake con rangos configurables
Número de manos jugadas como criterio
Porcentaje de rakeback variable según rendimiento
Aplicación automática mensual
